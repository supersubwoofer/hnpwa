Hacker News PWA

# Getting started

* copy or clone this repository
* change the names in package.json


## Install

```
* npm install --global gulp-cli
* npm install -g parcel-bundler
* npm install
```

## Build

```
npm run build
[ ] - !! bundle for browser
```

## Dev

```
* npm run dev

[ ] - improvement! watch src change then reload browser
```
